Five ways to import to PhyloSuite

Posted on 2019-03-01 | Edited on 2019-03-04 | In PhyloSuite demo

1. Search in NCBI

please see xxx.

2. Download by IDs

When you are in the PhyloSuite root folder, go to example\GenBank_entries_display_and_extract folder, open 104_flatworms_mtDNA_IDs.txt file to copy the IDs.

  1. Open the window via file-->Import file(s) or ID(s) in the menu bar;
  2. Enter IDs;
  3. Enter your email to tell NCBI who is downloading the sequences;
  4. Select a work folder to deposite the seuqences; selecting work folder of GenBank_File will download GenBank format seqence, selecting work folder of Other_File will download FASTA format seqence;
  5. Click Start.

3. Drag files to impo

When you are in the PhyloSuite root folder, go to example\GenBank_entries_display_and_extract folder:

  1. Select a work folder in GenBank_File root folder;
  2. Drag 21_flatworms_mtDNA.gb file and drop it in the display area.

You can also drag a fasta file, this way each sequence inside will be converted to GenBank format. Go to example\Convert_format folder, then drag cox1_NUC_mafft.fas file to the display area, the sequence name will be used as ID and Organism.

Sequence files with fasta, phylip, nexus formats and sequence recorded in word document can be dragged into the work folder in Other_File root folder:

4. Use button to import

  1. Open Input window via file-->Import file(s) or ID(s) in the menu bar, then click Choose Files to open files;

  2. When a work folder is empty, you can click Open file(s)/Input ID(s) in the display area to open files.

  3. When there are sequences in the work folder, you can use Add file option accessed via right-click menu.

5. Copy to import

  1. Select a work folder in GenBank_File root folder;

  2. Open 21_flatworms_mtDNA.gb file (see 3.3), CTRL+A to select all the contents, CTRL+C to copy, then CTRL+V to paste them in the display area.

  3. Similar to 2, you can also open cox1_NUC_mafft.fas (see 3.3) file to copy its contents to the display area.

  4. When you have a raw sequence, you can also copy it into the work folder.

  5. Reversely, if you select and CTRL+C to copy sequences in the display area, the GenBank format content will be copied to clipboard.
